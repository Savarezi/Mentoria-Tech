Bem-vindo(a) ao repositório do **Mentoria Tech**, uma solução inteligente e automatizada voltada para o desenvolvimento de carreira na área de tecnologia. Este projeto foi desenvolvido para oferecer um diagnóstico personalizado e um plano de ação estratégico para profissionais de tecnologia que buscam evolução constante.

---

## 💡  Projeto
O **Mentoria Tech** é um assistente de carreira (chatbot) projetado para coletar dados, analisar competências comportamentais e técnicas, e gerar recomendações personalizadas. A ideia central é atuar como um mentor digital que utiliza lógica de dados para estruturar o futuro profissional do usuário.

## 🛠️ Tecnologias e Ferramentas
*   **Fluxo de Automação:** Estruturado no typebot para criação de jornada de usuário.
*   **Inteligência Artificial:** Processamento de dados v(integrado para análise estratégica).
*   **Metodologias:** Foco em qualidade, processos, análise de dados e segurança da informação.
*   **Design de Experiência:** Focado em tom humanizado, empático e estratégico.

## ⚙️ Funcionalidades
*   **Diagnóstico de Perfil:** Coleta de dados como metas de 12 meses/5 anos e conhecimentos técnicos (Python, SQL, AWS, n8n, etc.).
*   **Soft Skills:** Avaliação estruturada de liderança, comunicação, resolução de problemas e trabalho em equipe.
*   **Mentoria Estratégica:** Processamento das variáveis coletadas para entrega de um plano de ação detalhado para os próximos 3 meses.
*   **Personalização:** Cada mentoria é adaptada ao nível de senioridade e área de interesse do usuário.

## 🚀 Como Funciona
1.  **Coleta:** O bot guia o usuário por blocos lógicos, armazenando respostas em variáveis.
2.  **Análise:** O sistema processa os dados do perfil (conhecimentos + metas + soft skills).
3.  **Entrega:** A IA gera um feedback estruturado contendo:
    *   **Validação:** Reconhecimento da trajetória atual.
    *   **Análise Crítica:** Conexão entre o perfil técnico e o cargo desejado.
    *   **Plano de Ação:** 3 passos práticos para evolução imediata.
